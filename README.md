***DemoExam*** - Проект, выполненный в рамках примера демонстрационного экзамена Профильного уровня внутри Государственной Итоговой Аттестации по специальности 09.02.07 2026 г. (Квалификация - Программист)

## Важно!
> [!Important]
> Проект выполнен в качестве примера для тренировки и не имеет цели нарушить чьи-то авторские права. Исходные данные для разработки проекта взяты из официальных открытых источников

## Предметная область - Магазин обуви

## Структура проекта
```
├───DemoExamSolution # Проект 
│   ├───AdditionalWindows
│   ├───DTO
│   ├───Entities
|   ├───Resources
│   └───RoleWindows
├───src # Ресурсы для разработки
│   ├───Module1 # Данные для импорта (приведенные), картинки, описание предметной области
│   ├───Module2 # Руководство по стилю 
│   └───Inputs # Выходные данные (для некоторых модулей)
├───Docs # Прочие графические документы (ER-диаграмма, алгоритм, результаты работы)
└───Script.sql # Скрипт со схемой и данными 
```
